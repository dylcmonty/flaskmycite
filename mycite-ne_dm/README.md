# mycite-ne_dm

Demo/user natural-entity portal instance (DM).

## Run locally

```bash
cd mycite-ne_dm
source .venv/bin/activate
python app.py
```

Note:

- `app.py` default port is `5000`.

## Scope

- Active portal app instance with alias-session UI and aliases API wiring.
